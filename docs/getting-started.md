# Getting Started

## 1. Install Yanzi

See [Install](install.md).

## 2. Create a Project

```bash
yanzi project create demo
yanzi project use demo
```

## 3. Capture an Interaction

```bash
yanzi capture --author "Ada" --prompt "hello" --response "world"
```

## 4. Export HTML

```bash
yanzi export --format html
```

## 5. Open the UI

```bash
open YANZI_LOG.html
```
