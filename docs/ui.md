# Yanzi UI

The UI is the HTML file generated by `yanzi export --format html`.
It is not a separate application.

It exists to visualize stored AI artifacts over time in a readable format.

## Features

- Collapsible artifacts
- Prompt preview
- Copy buttons
- Sticky header
- Search
- Checkpoint boundaries

## Usage

```bash
yanzi export --format html
```

## Philosophy

- The UI is a projection of stored artifacts
- No hidden state
- Fully deterministic
