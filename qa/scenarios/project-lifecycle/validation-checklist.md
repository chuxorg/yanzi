# Validation Checklist

- [ ] Project is created or bound successfully.
- [ ] Deterministic capture input files are recorded.
- [ ] Capture operation exits `0` and produces retrievable artifact.
- [ ] Artifact list confirms captured record visibility.
- [ ] Checkpoint creation exits `0`.
- [ ] Checkpoint list confirms checkpoint visibility.
- [ ] Rehydrate exits `0` and restores expected state.
- [ ] Certification outcome and evidence are written to report.
