# Validation Checklist

- [ ] Candidate tag is explicitly provided.
- [ ] Installer pinned install succeeded for candidate tag.
- [ ] Installed runtime version matches candidate tag.
- [ ] Uninstall removed executable from path.
- [ ] Reinstall restored same candidate lineage.
- [ ] Homebrew lineage evaluated against candidate tag.
- [ ] PASS/WARN/FAIL classification recorded with findings.
