# Validation Checklist

- [ ] Candidate tag and SHA recorded.
- [ ] Installer executed with explicit `--version <candidate-tag>`.
- [ ] Installed runtime version matches candidate tag.
- [ ] Installer provenance recorded.
- [ ] Homebrew tap formula version and checksum verified against candidate policy.
- [ ] Homebrew installed runtime version recorded.
- [ ] Uninstall/reinstall checks completed.
- [ ] PASS | WARN | FAIL decision recorded with release recommendation.
